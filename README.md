# docker-wordpress
> Docker for wordpress.

## get-started
```shell
# common manngeent
./bin/start.sh
./bin/stop.sh
./bin/restart.sh

## be careful!!!
./bin/reset.sh
```

## setting
~~~
cd docker-wordpress
sudo chown -R www-data:www-data ./data/wp_data/wp-content
~~~
